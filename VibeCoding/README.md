# StudyMate AI

A web-based AI study assistant that helps users summarize notes, explain concepts, and generate flashcards.

## Features

*   **AI-powered text processing:** Summarize, explain, and generate flashcards from any text.
*   **User authentication:** Secure user accounts with JWT-based authentication.
*   **Study history:** Keep track of your past study sessions.
*   **Responsive design:** Works on all devices.

## Tech Stack

*   **Frontend:** HTML, CSS, JavaScript, Bootstrap
*   **Backend:** PHP, MySQL
*   **AI:** Gemini Pro

## How to run locally

1.  Clone the repository.
2.  Create a `.env` file with your database credentials and API keys.
3.  Run `composer install`.
4.  Run `php -S localhost:8000`.
